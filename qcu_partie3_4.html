Exercice : Question 6

Exprimer Eint en fonction de Erk et TLk. En langage Python ou C++ ou en pseudocode, rédiger le corps d’une fonction calcul_eclairement qui prend en arguments les listes T_calcules et mesures et renvoie la valeur de l’éclairement intérieur de la cabine.

Votre choixChoix attenduRéponse

  • \(E_{\text{int}} = \sum_{k=0}^{\text{3}} E_{rk} \cdot TL_k\)

  • \(E_{\text{int}} = \sum_{k=7}^{\text{1}} E_{rk} \cdot TL_k\)

  • \(E_{\text{int}} = \sum_{k=0}^{\text{3}} TL_k \cdot E_{rk}\)

En Python :

Eint = 0

for k in range(4):

Eint = Eint + mesures[k][0] * T_calcules[k]

return Eint

En C++ :

float Eint = 0 ;

for (k = 0 ; k < 4 ; k = k + 1) Eint = Eint + mesures[k][0] * T_calcules[k] ;

return Eint ;

.

En pseudo-code :

Eint ← 0

Pour k de 0 à 3 faire Eint ← Eint + mesures[k][0] * T_calcules[k]
